Pedidos de suporte

  • Crie local, pessoas e tipos de turno antes da escala.
  • Use a visão Eu ou filtros para ver turnos e colegas.
  • Revise o reconhecimento da imagem antes de salvar.
  • Permita notificações para receber lembretes.
  • Exclua locais, pessoas, turnos, regras e dados relacionados no app.

Informações armazenadas

Os dados são locais; com iCloud privado, Apple sincroniza dados compatíveis. RosterSlate não opera servidor próprio.

Política de Privacidade

Não é software médico

Ferramenta geral que não guarda pacientes, dados clínicos, prontuários, receitas ou decisões médicas.
